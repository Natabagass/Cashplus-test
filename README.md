
# Friendchise 📝  


## Get Started 🚀  
Clone project ini 

~~~bash  
  git clone https://github.com/Natabagass/Cashplus-test
~~~

Kemudian navigasi ke project directory 

~~~bash  
  cd Cashplus-test
~~~

Kita juga harus menginstall dependencies

~~~bash  
npm install
~~~

Memulai server 

~~~bash  
npm run dev
~~~  

## Fitur 🔥  
- Login & Registrasi Pengguna
- Mencari Produk
